The Wumpus 2014 Experience

Welcome, Wumpus is the classic basic game from 1972, now in JavaScript. Hunt the Wumpus in the cave using the arrow keys. You can shoot with the keys W=up, S=down, A=left, D=right, but be careful, you only have 8 arrows. The treasure is gold-colored and worth 20 bananas, take it!!!
